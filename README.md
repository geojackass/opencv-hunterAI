# opencv-hunterAI
Docker container for opencv and tensorflow

### prerequisties
VirtualMachine Memory 4G or more

### usage
- build container from Dockerfile
```
docker-compose build
```
- UP(RUN)
```
docker-compose up -d
```

- login from host to docker
```
docker-compose exec opencv_hunter_ai bash
```

- USE Port 8080
```
localhost(IP):8080
```
